* Matrices

** Transpose

   #+BEGIN_SRC python
     def transpose(matrix):
         """Calculate A^T on a 2D array."""
         list(zip(*matrix))

** Rotate 90 Degrees

   #+BEGIN_SRC python
     def rotate_clockwise(matrix):
         """Rotate a matrix 90 degrees with flip(A)^T."""
         return list(zip(*matrix[::-1]))

     def rotate_anticlockwise(matrix):
         """Rotate a matrix -90 degrees with flip(A^T)."""
         return list(zip(*matrix))[::-1]
   #+END_SRC
